PROJECT DESCRIPTION:

Create music with your mind.

The Muse 2 Headset sends EEG data—where each of the four sensors gives your 
γ, β, α, θ, δ brain waves in Hz—to the Mind Monitor mobile app, and it is then streamed to VS Code. 
It uses those values to generate music and visuals reflective of the user's emotional state.
